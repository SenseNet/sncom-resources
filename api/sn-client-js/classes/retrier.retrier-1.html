Class Retrier

Utility class for retrying operations. Usage example:

         const funcToRetry: () => Promise<boolean> = async () => {
             let hasSucceeded = false;
             // ...
             // custom logic
             // ...
             return hasSucceeded;
         }
         const retrierSuccess = await Retrier.Create(funcToRetry)
             .Setup({
                 retries: 3,
                 retryIntervalMs: 1,
                 timeoutMs: 1000
             })
             .Run();
